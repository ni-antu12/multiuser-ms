# Assets

Esta carpeta debe contener los siguientes archivos de imagen:

## Archivos requeridos:

1. **icon.png** (1024x1024) - Ícono de la aplicación
2. **splash.png** (1284x2778) - Pantalla de inicio
3. **adaptive-icon.png** (1024x1024) - Ícono adaptativo para Android
4. **favicon.png** (48x48) - Favicon para web

## Notas:

- Puedes generar estos assets usando el [Expo Asset Generator](https://www.appicon.co/)
- O usar placeholders temporales hasta que diseñes los íconos finales
- Los archivos deben estar en formato PNG con fondo transparente (excepto splash)

## Placeholders temporales:

Mientras tanto, la app funcionará sin estos archivos, pero se recomienda agregarlos antes de publicar.
